Shader Schedule Reminder is a sleek, TypeScript-powered web app designed to help users manage daily tasks with clarity and ease. It features a clean, responsive UI that adapts across devices, making it ideal for students and professionals alike. Users can add, edit, and delete reminders, with visual cues and time-based alerts to stay on track. Built with modular components and modern styling, it ensures both performance and maintainability. Whether for class schedules or personal goals, Shader keeps your day structured and stress-free.
  # Shader Reminder

  This is a code bundle for Shader Reminder. The original project is available at https://www.figma.com/design/VjYExsap3jxpuv7JfqVTL5/Shader-Reminder.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
